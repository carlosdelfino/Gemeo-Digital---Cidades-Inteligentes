Sempre use a pasta venv, vá buscando de forma descendente até o raiz do projeto.

Nunca instale no sistema operacional algum pacote para uso nos scripts.

requirements.txt deve ficar na past scripts

Cada script python deve ter seu lançador com o mesmo nome, para bash, bat e ps1.

Sempre que houver complexidade no processo de tratamento dos dados, use threads, cuda ou OpenMP, use apenas 50% dos recusos disponíveis para que a maquina não trave.
